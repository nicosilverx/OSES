\doxysubsection{Detailed Description}
The Trigger Control Block contains\+:
\begin{DoxyItemize}
\item the content of the Alarm Control Block if only Alarms are present in the current application.
\item the content of the Schedule Table Control block if only Schedule Tables are present in the current application.
\end{DoxyItemize}

If both Alarms and Schedule Tables are present, the information above are linked through the Trigger Descriptor Block to the Alarm/\+Schedule Table Descriptor Block to the Alarm/\+Schedule Table control block.
