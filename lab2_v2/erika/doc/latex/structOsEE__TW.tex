\doxysubsection{Detailed Description}
A Task wrapper is useful every time some actions have to be performed before we need to execute the task. For example, it is useful on some architecture\textquotesingle{}s system timer to perform actions on the peripheral registers before calling the I\+S\+R2 (remember that I\+S\+R2 and Tasks in E\+R\+I\+K\+A3 are handled in a similar way)
