◆ osEE_sn_priority_insert()

OsEE_bool osEE_sn_priority_insert ( OsEE_SN **  pp_first,
OsEE_SN *  p_sn_new,
CONST(OsEE_bool,)  as_ready 
)

This functon inserts a task inside a task queue, following the order of the task.

Parameters
[in,out]pp_firstPointer to (the pointer to the first node of a queue)
[in]p_sn_newNode to be inserted in the list following the priority of its task
[in]as_readyIf OSEE_TRUE, use the ready_priority and not the current priority to queue the task. The usage of the ready priority is useful when doing global schedulng. In AUTOSAR conformance classes, onlt the current priority is used.
Returns
The function returns OSEE_TRUE if the head pp_first changed.
