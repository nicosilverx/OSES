Detailed Description

This data structure is the "starting point" for navigating all the kernel data structures. It contains static flash information about all the global objects included in the system, such as:

  • cores
  • tasks
  • resources
  • counters
  • alarms
  • scheduling tables
  • spin locks
  • barriers
