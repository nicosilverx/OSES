Detailed Description

A Task wrapper is useful every time some actions have to be performed before we need to execute the task. For example, it is useful on some architecture's system timer to perform actions on the peripheral registers before calling the ISR2 (remember that ISR2 and Tasks in ERIKA3 are handled in a similar way)
